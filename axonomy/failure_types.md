# Failure Taxonomy

## Hallucination
The model invents facts, laws, statistics, or sources.

## Overconfident Incorrect Answer
The model provides a wrong answer with high confidence.

## Unsafe Advice
The model provides guidance that could cause harm
(e.g., medical, legal, financial).

## Scope Violation
The model acts beyond its intended role or authority.

## False Authority Claim
The model implies professional qualification or certainty
it does not have.
